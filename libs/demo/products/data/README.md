# Products Data

This library contains data access services for the products domain.

## Usage

Import the service in your feature components:

```typescript
import { ProductsService } from "@demo/products/data";
```

## Structure

- `src/lib/products.service.ts` - Main data service
- `src/lib/products.service.spec.ts` - Service tests
- `src/public-api.ts` - Public API exports
