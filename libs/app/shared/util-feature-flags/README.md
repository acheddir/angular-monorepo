# Util FeatureFlags

This library contains utility functions for the shared domain.

## Usage

Import utilities in your components and services:

```typescript
import { featureFlags } from "@app/shared/util-feature-flags";
```

## Structure

- `src/lib/feature-flags.ts` - Utility functions
- `src/lib/feature-flags.spec.ts` - Utility tests
- `src/public-api.ts` - Public API exports
